HTML Giriş

HTML, Web sayfaları oluşturmak için standart biçimlendirme dilidir

HTML Nedir?

  • HTML, Hiper Metin İşaretleme Dili anlamına gelir.
  • HTML, Web sayfaları oluşturmak için standart işaretleme dilidir.
  • HTML, bir Web sayfasının yapısını tanımlar.
  • HTML, bir dizi elementten oluşur.
  • HTML elementleri, tarayıcıya içeriği nasıl göstermesi gerektiğini söyler.
  • HTML elementleri, "başlık", "paragraf", "bağlantı" gibi içerik parçalarını etiketler.

Alıştırma

<!DOCTYPE html> <html> <head> <title>Sayfa başlığı</title> </head> <body> <h1>Bu bir başlık.</h1> <p>Bu bir paragraf.</p> </body> </html>
