HTML Style

HTML style özelliği, bir öğeye renk, yazı tipi, boyut ve daha fazlası gibi stiller eklemek için kullanılır.

HTML Style Özelliği

Bir HTML öğesinin stilinin ayarlanması style özelliğiyle yapılabilir.

HTML style özelliği aşağıdaki sözdizimine sahiptir:

<tagname style="property:value;">

Özellik bir CSS özelliğidir. Değer bir CSS değeridir.

Bu eğitimin ilerleyen kısımlarında CSS hakkında daha fazla bilgi edineceksiniz.

Arkaplan Rengi

CSS background-color özelliği, bir HTML öğesinin arka plan rengini tanımlar.

<body style="background-color:powderblue;">
<h1 style="background-color:powderblue;"> Başlık </h1>
<p style="background-color:tomato;"> Paragraf </p>
